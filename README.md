# KNN para Predição de Diabetes

## Descrição

Este projeto implementa um modelo de Machine Learning utilizando o algoritmo **K-Nearest Neighbors (KNN)** para classificação de pacientes quanto ao diagnóstico de diabetes.

O desenvolvimento foi realizado em **R Markdown (Rmd)** e contempla todas as etapas do processo de aprendizado de máquina, desde o pré-processamento dos dados até a avaliação do modelo.

## Funcionalidades

- Importação do conjunto de dados
- Conversão das variáveis categóricas em fatores
- Divisão dos dados em treino, validação e teste
- Padronização das variáveis numéricas
- Construção do modelo KNN
- Avaliação do desempenho do modelo

## Tecnologias utilizadas

- R
- R Markdown
- caret
- dplyr
- readr

 
## Base de dados

O projeto utiliza um conjunto de dados contendo informações clínicas e demográficas de pacientes para previsão do diagnóstico de diabetes.

Caso a base de dados não esteja disponível no repositório, adicione o arquivo `diabetes_dataset.csv` na pasta do projeto e ajuste o caminho do arquivo no código, se necessário.
